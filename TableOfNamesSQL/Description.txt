Проект TableOfNamesSQL------------------------------------------------------------------------------------------
Авторы: Бычков Д.А., Торопов М.Е.

В этом проекте реализовано создание таблицы имен для популярных реляционных БД.

Руководство по использованию проекта:
	
1) Для компиляции требуется:
	- в менеджере ссылок проекта указать ссылки на библиотеки: 
	PolarDB.dll
	System.Data.SQLite.dll
	MySql.Data.dll
	Они находятся в папках "Common\DLLs" и "Common\DLLs64"

2) Описание классов:
	- Testing - основной класс, в котором описаны методы для загрузки и поиска данных.
	- MySQL, MSSQL, SQLite - классы, содержащие методы для создания соответствующих БД, создания индексов, методы поиска и добавления данных.
	- SQLDB - класс-шаблон для классов, реализующих интерфейс взаимодействия с SQL БД

Сделано:
- Программное создание и удаление БД
- Измерение характеристик:
	* Добавление данных
	* Построение индексов
	* Поиск строки по id
	* Поиск id по строке
- Вывод результатов в файл
